What is Click to Chat?

WhatsApp's Click to Chat feature allows you to begin a chat with someone without having their phone number saved in your phone's address book. As long as you know this personâ€™s phone number, you can create a link that will allow you to start a chat with them. By clicking the link, a chat with the person automatically opens. Click to Chat works on both your phone and WhatsApp Web.
For a detailed explanation on international numbers, read this article.

How to Use

1. Enter the phone number of the recipient in International Format.
2. If you don't want to include a message, skip to point 5.
3. If you want to include a message, check the Add Message checkbox.
4. Enter the message you want to send.
5. Click on "Click-To-Chat" button.
6. When asked if you want to be redirected, select 'Ok' or 'Yes'.
